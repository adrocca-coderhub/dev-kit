# Feature Workflow

## Objetivo
Implementar nuevas funcionalidades con claridad de alcance, contexto, documentación e impacto.

## Flujo recomendado
1. Leer requerimiento.
2. Hacer análisis profundo.
3. Identificar alcance y dependencias.
4. Diseñar solución.
5. Implementar.
6. Documentar Jira si aplica.
7. Documentar PR.
8. Cerrar con checklist de entrega.
