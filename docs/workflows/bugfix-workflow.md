# Bugfix Workflow

## Objetivo
Abordar bugs de forma estructurada, reduciendo retrabajo y mejorando claridad técnica.

## Flujo recomendado
1. Entender síntoma y contexto.
2. Comparar comportamiento esperado vs actual.
3. Recolectar evidencia.
4. Formular hipótesis.
5. Validar causa raíz.
6. Implementar corrección.
7. Documentar impacto.
8. Preparar PR o Jira si aplica.
