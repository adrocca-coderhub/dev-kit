# Data Task Workflow

## Objetivo
Estandarizar tareas de calidad, preprocesamiento, transformación y carga de datos.

## Flujo recomendado
1. Perfilar fuente de datos.
2. Validar esquema.
3. Preprocesar datos.
4. Transformar según reglas de negocio.
5. Validar output.
6. Cargar al destino.
7. Documentar input/output y riesgos.
