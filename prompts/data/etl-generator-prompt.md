# ETL Generator Prompt

Actúa como una ingeniera de datos senior.

Necesito diseñar o generar una base robusta de ETL.

## Objetivo
Crear un flujo ETL modular, claro, mantenible y reutilizable.

## Debe incluir
- etapa de extracción
- validación del input
- preprocesamiento
- transformación
- validación de reglas de negocio
- carga
- logging
- manejo de errores
- configuración por parámetros
- resumen final de ejecución
- ejemplos de input y output

## Reglas
- Separar responsabilidades por etapa.
- Explicar qué hace cada parte.
- Si no hay suficiente contexto, generar una base genérica limpia.
- Incluir estructura de archivos sugerida.
- Incluir funciones mínimas recomendadas.
- Incluir recomendaciones de mejora y extensibilidad.

## Formato esperado
1. Resumen
2. Estructura propuesta
3. Flujo ETL
4. Funciones principales
5. Configuración
6. Ejemplo de input
7. Ejemplo de output
8. Errores esperados
9. Recomendaciones
