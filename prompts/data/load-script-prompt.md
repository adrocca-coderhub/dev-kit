# Load Script Prompt

Actúa como una ingeniera de datos senior.

Necesito diseñar o generar un script base de carga de datos.

## Objetivo
Construir una etapa de carga segura, clara, reutilizable y bien documentada.

## Debe incluir
- validación previa del dataset transformado
- validación del esquema destino
- estrategia de carga
- manejo de inserción, append, update o upsert según corresponda
- logging
- manejo de errores
- métricas de carga
- resumen final
- ejemplos de input y output

## Reglas
- Explicar el destino final.
- Definir claramente la estrategia de carga.
- Incluir errores controlados.
- Describir resultado esperado.
- Mantener separación respecto al preprocesamiento y transformación.

## Formato esperado
1. Resumen
2. Destino final
3. Estrategia de carga
4. Validaciones
5. Flujo de carga
6. Errores y edge cases
7. Output esperado
8. Recomendaciones
