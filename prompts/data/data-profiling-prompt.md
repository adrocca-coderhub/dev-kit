# Data Profiling Prompt

Actúa como una especialista en análisis y perfilado de datos.

Necesito evaluar una fuente de datos o esquema antes de procesarlo.

## Objetivo
Detectar problemas de calidad, estructura, tipado y consistencia para proponer reglas de preprocesamiento y validación.

## Debes analizar
- columnas o campos
- tipos aparentes
- nulos
- duplicados
- formatos inconsistentes
- valores inválidos
- columnas obligatorias
- riesgos de calidad
- reglas de limpieza sugeridas
- output esperado tras limpieza

## Formato esperado
1. Resumen del dataset
2. Hallazgos principales
3. Tabla de campos
4. Riesgos detectados
5. Reglas de preprocesamiento sugeridas
6. Recomendaciones
7. Ejemplo de esquema limpio esperado
