# Dev Kit

Kit personal para acelerar trabajo de desarrollo fullstack, documentación técnica y automatización de tareas repetitivas.

## Objetivo
Centralizar templates, prompts, skills, snippets, scripts base y playbooks reutilizables para trabajar más rápido, con mejor calidad y mayor consistencia.

## Estructura

| Carpeta | Propósito |
|---|---|
| `prompts/` | Prompts reutilizables para IA |
| `templates/` | Moldes base para documentación y código |
| `skills/` | Instrucciones estructuradas por tipo de tarea |
| `playbooks/` | Guías operativas y checklists |
| `scripts/` | Scripts base reutilizables |
| `snippets/` | Snippets para VS Code |
| `examples/` | Ejemplos de input, output y documentación |
| `.opencode/commands/` | Comandos reutilizables para OpenCode |

## Casos de uso
- redactar documentación de PR
- redactar tarjetas Jira robustas
- analizar proyectos o módulos
- generar scripts base de datos / ETL
- reutilizar snippets y templates
- estandarizar entregables

## Primeros pasos
1. Revisar `prompts/master/prompt-master.md`
2. Revisar `skills/`
3. Revisar `templates/`
4. Usar `examples/` como referencia
5. Ajustar `snippets/` a VS Code
6. Probar comandos en `.opencode/commands/`

## Recomendación de uso
No usar todo a la vez.
Tomar solo lo necesario para cada proyecto y personalizarlo.
