description: Run a deep repository or feature analysis
agent: plan
---

Perform a deep technical and functional analysis of the current repository or feature.

Do not modify files.

Analyze:
1. repository purpose
2. architecture and module boundaries
3. main execution flows
4. key files and responsibilities
5. external dependencies and integrations
6. configuration and environment requirements
7. data flow
8. business rules inferred from code
9. technical debt and risks
10. missing documentation
11. reusable patterns that could be extracted into templates
12. suggested next actions for a junior fullstack developer

Deliver the result in Markdown with:
- executive summary
- architecture overview
- file/module map
- flow explanation
- risks and gaps
- improvement opportunities
- concrete action plan
